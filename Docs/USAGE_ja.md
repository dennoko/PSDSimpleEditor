# PSD Simple Editor 使用方法

PSD Simple Editorは、Unityエディタ上でPSDファイルを読み込み、レイヤーの表示設定や色調補正の編集を行って、最終結果をPNGまたはPSD形式で書き出すことができるツールです。

## ウィンドウの起動方法
Unityのメニューバーから、以下の項目を選択してウィンドウを開きます。
- dennokoworks -> PSD Simple Editor

## PSDファイルの読み込み
1. ウィンドウ上部にある「PSD:」入力欄に、読み込みたいPSDファイルのパスを指定します。
   - Windowsのエクスプローラーからコピーしたパス（前後にダブルクォーテーション `"` が付いた状態）をそのまま貼り付けても、自動的にトリムされて読み込まれます。
   - 右側の「Browse...」ボタンをクリックして、ファイル選択ダイアログからファイルを選択することもできます。
2. 入力後、「Load」ボタンをクリックするとPSDファイルが解析され、レイヤー一覧とプレビューがウィンドウ内にロードされます。

## 保存・出力先の設定
ウィンドウ上部にある「Export Dir:」入力欄に、画像を書き出す出力先フォルダのパスを指定します。
- Unityプロジェクト内のフォルダ（Assets/...形式の相対パス）またはPC内の絶対パスのどちらでも指定可能です。
- 右側の「Browse...」ボタンをクリックして、フォルダ選択ダイアログからフォルダを指定することもできます。

## レイヤーパネルの操作（画面左側）
PSDファイルに含まれるすべてのレイヤーおよびグループがツリー形式で一覧表示されます。
- ツリーは一番上が「最上層」となる順序で描画されます。
- レイヤー名の左端にあるチェックボックスを切り替えることで、各レイヤーやグループの表示（オン）/ 非表示（オフ）を切り替えられます。
- グループ（フォルダ）名の左側にある折りたたみボタンをクリックすることで、グループ内のレイヤーを展開・格納できます。
- レイヤー名やグループ名の右側にあるドロップダウンメニューから、各要素のブレンドモード（通常、乗算、スクリーン、オーバーレイなど。グループの場合はPassThroughも可）を切り替えることができます。
- 表示中のレイヤーまたはグループを展開すると、以下のパラメータ編集項目が表示されます。
  - 不透明度: スライダーをドラッグして不透明度を 0.0〜1.0 の範囲で変更します。
  - 色調補正（ピクセルレイヤー用）: 「色調補正」フォールドアウトを展開すると、以下の調整スライダーが表示されます。
    - 明るさ（-150〜150）
    - コントラスト（-50〜100）
    - 色相（-180〜180）
    - 彩度（-100〜100）
    - 明度（-100〜100）
  - グラデーションマップ（ピクセルレイヤー用）: 「グラデーションマップ」チェックボックスをオンにすると有効化され、以下が設定できます。
    - 階調: クリックするとグラデーションエディタが開き、グラデーションの色を編集できます。
    - 適用率: グラデーションの適用率をスライダーで 0.0〜1.0 の範囲で調節します。
  - 調整レイヤーおよびベタ塗り（SoCo）レイヤー:
    - PSDファイル内で作成された「明るさ・コントラスト調整」「色相・彩度・明度調整」「ベタ塗り」のレイヤーは、レイヤーパネル上に直接調整コントロール（スライダーやカラーピッカー）が表示され、その場で値を変更できます。
- レイヤーパネルと右側のプレビューパネルの間にある縦の境界線（スプリッター）は、左右にドラッグしてパネルの幅を調節することができます。

## プレビュー表示（画面右側）
- レイヤーパネルでの編集（表示・非表示の変更、不透明度やブレンドモードの調整、色調補正パラメータの変更など）がリアルタイムに合成され、プレビューエリアに反映されます。
- 透明部分は市松模様（チェッカー背景）で描画されます。
- ウィンドウ上部の「マージ参照」トグルボタンをオンにすると、プレビュー画面の右下にPSDファイル内に格納されている「マージ済み画像（Photoshop上で保存された元の最終画像）」がサムネイルとして重なって表示されます。本ツールによる再合成結果と元の画像の見た目を比較したい場合に便利です。

## ステータス表示と画像の書き出し（画面下部）
ウィンドウの最下部に、読み込んでいるPSDファイルの「幅 × 高さ」「レイヤー数」「ビット深度」「カラーモード（RGB、Grayscaleなど）」の情報が表示されます。

- Export PNG ボタン:
  現在のプレビュー合成結果をPNG画像として書き出します。書き出し先は「Export Dir」で設定したフォルダになります。
  - Unityプロジェクト内（Assets以下）へ保存した場合は自動的にアセットとして検出され、プロジェクトビュー内で保存されたPNGファイルがハイライトされます。
  - プロジェクト外へ保存した場合は、自動的にエクスプローラーで保存先フォルダが開きます。

- Export PSD... ボタン:
  現在のレイヤーの表示状態や調整パラメータの値を維持した状態で、新しく別のPSDファイルとして書き出すための保存ダイアログを開きます。
